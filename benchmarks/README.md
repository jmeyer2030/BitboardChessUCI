# Benchmarks

Standalone benchmark suites for the DriftWood chess engine. These are **not** JUnit tests — they live outside `src/test` and are run independently.

| Benchmark | Directory | Description |
|---|---|---|
| Puzzle Bench | [`puzzle-bench/`](puzzle-bench/) | Tests the engine against sampled Lichess puzzles and reports solve accuracy by rating bucket. |

See each benchmark's own `README.md` for usage and configuration.
